What are FOR loops?

We have seen FOR blocks before

We used a FOR loop in the last project, we did in the lesson walkaround1. The cool kid moved along the path.Without the loop we would have to work out how far wee wanted the cool kid to go and we could only run that bit of code once.

Remember the variables from the last lesson we are using them here. The variable starts with the number 1 stored in it. Every time the loop goes around we add 1 to it so the kid moves 1 step to the right on each FOR loop.
