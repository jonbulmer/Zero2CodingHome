What are variables

A variable is a place in a computers memory where we can store information. This is only as long as the program is running. You should try and think of different names for your variables that give you a good idea of what they are for.

If you want to keep a count of goals scored call your variable 'GoalsScored'.

There are 2 very basic ways to store information in our variables letters and numbers. We have a word for information we store in a variable and the word is 'Data'. Data stored as letters has a name 'String'. Like a piece of string that you can join another piece of string to it by tying a knot. The second data type we are going to look at is numbers. The name we give it is numeric. There are other data types and we will look at them later.
